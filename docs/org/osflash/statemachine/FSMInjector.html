Packageorg.osflash.statemachine
Classpublic class FSMInjector

Creates and registers a StateMachine described from a data source.

This allows reconfiguration of the StateMachine without changing any code, as well as making it easier than creating all the IState instances and registering them with the StateMachine at startup time. The FSMInjector delegates the knowledge of the data format to its IStateDecoder
